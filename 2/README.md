# How to Start
## install packages
<code>npm install</code>
## start
<code>npm start</code>
------
## set node 2
<code>$env:HTTP_PORT=3002    <p>
$env:P2P_PORT=6002    <p>
(optional) $env:PEERS = "ws://127.0.0.1:6001"</code>
------
